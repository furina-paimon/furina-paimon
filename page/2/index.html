Hexo GUI 介绍

概述

File-Structure-Diagram

概述

File-Structure-Diagram 是一个多功能的目录结构扫描与可视化工具。它能够递归地扫描指定目录下的所有子目录和文件，并以清晰的树状图形式展示文件和目录的层级关系。该工具还支持自动排除超过指定大小的目录、用户自定义排除目录、实时进度显示以及将结构导出为文本文件等功能。

核心功能

1. 目录扫描模块

负责递归扫描指定根目录及其所有子目录，计算每个目录的大小，并获取文件和目录的基本信息，支持深度扫描，确保不遗漏任何文件或目录。

2. 结构可视化模块

将扫描结果以树状图形式展示，使用特殊符号标记目录层级关系，使文件结构一目了然，支持多级目录显示，确保复杂结构也能清晰呈现。

3. 自动排除模块

可根据设定的大小阈值，自动排除超过该大小的目录，提高扫描效率，适用于处理大型项目目录。

4. 用户交互模块

提供友好的命令行界面，用户可通过输入索引灵活选择要排除的目录，界面简洁直观，操作便捷。

5. 导出功能模块

将构建的文件结构索引导出为文本文件，方便用户保存、分享和备份目录结构信息，导出文件格式清晰，易于阅读。

6. 实时监控模块

在扫描过程中实时显示进度条，直观反映扫描进度，帮助用户了解当前扫描状态和剩余时间。
